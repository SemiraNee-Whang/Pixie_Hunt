# Pixie Treasure Hunt

**Description:**  
A 2D treasure-hunting adventure game in Java with cute, pastel visuals and lofi vibes. Explore whimsical islands, solve puzzles, and collect magical treasures.

---

## Features

- Explore colorful islands and hidden areas
- Solve mini-puzzles to unlock treasures
- Collect gems, outfits, and decorations
- Relaxing lofi music and pastel aesthetic

_(More features will be added as the game develops)_

---
